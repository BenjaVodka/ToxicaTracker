# 🕵️‍♂️ ToxicTracker (TóxicaTracker)

**ToxicTracker** is a professional privacy-focused tool to analyze your Instagram followers and following without ever sharing your password. Discover who unfollowed you, who doesn't follow you back, and track your digital circle with 100% local data processing.

## 🏗️ Project Structure

This is a professional monorepo setup:

- `/frontend`: React + Vite + TailwindCSS (Modern, glossy UI)
- `/backend`: Spring Boot + JPA + H2 (Robust analysis engine)
- `/docs`: Project documentation and architecture details (coming soon)

## 🚀 Getting Started

### Prerequisites

- **Node.js**: v18 or higher
- **Java**: JDK 17
- **Maven**: (mvnw included in backend)

### Setup & Run

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run All Services**:
   ```bash
   # From the root directory
   npm run dev:frontend
   npm run dev:backend
   ```

## 🛡️ Privacy First (Cybersecurity)

TóxicaTracker is built with a security-first mindset. Unlike other apps that request your Instagram credentials, we only process local JSON data exports. Your account is never at risk, and your data never leaves your device.

## 🛠️ Technology Stack

- **Frontend**: React, Framer Motion, Lucide React, TailwindCSS, Vite.
- **Backend**: Spring Boot 3, Java 17, Spring Data JPA, Lombok.
- **Database**: H2 (In-memory for development/MVP).
